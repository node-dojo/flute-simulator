# FLUTE SIMULATOR

An interactive web-based Native American Flute simulator with keyboard controls and visual feedback.

## Features

- **Interactive Flute**: Click holes or use keyboard to play notes
- **Multiple Keys**: Select from A2 to G#5 (3.5 octaves)
- **Keyboard Controls**: 
  - `A, S, D, J, K, L` - Close/open holes (from high to low)
  - `SPACEBAR` - Blow/play the note
- **Traditional Lock**: Option to lock the 3rd hole closed (traditional style)
- **Dark Mode**: Black and white inverted color scheme
- **Pentatonic Minor Scale**: Authentic Native American flute scale

## How to Play

### Keyboard Method
1. Press `A, S, D, J, K, L` to close holes (from highest to lowest note)
2. Hold `SPACEBAR` to blow and play the note
3. All 6 keys + space = Base note (fundamental)
4. Fewer keys = Higher notes

### Click Method
- Click any hole to play that specific note

## Technical Details

- Built with vanilla HTML, CSS, and JavaScript
- Uses Web Audio API for sound generation
- Responsive design with custom fonts (LED Dot-Matrix for title, Source Code Pro for body)
